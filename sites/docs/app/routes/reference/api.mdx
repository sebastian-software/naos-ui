# API Reference

This page records the v0.1 public API shape. Rust crates remain internal and are
not published to crates.io for v0.1.

## `@iktia/core`

| Export | Stability | Notes |
| --- | --- | --- |
| `state()` | Public | Writable local state primitive. |
| `computed()` | Public | Derived read-only value. |
| `effect()` | Public | Lifecycle-aware effect and cleanup. |
| `event()` | Public | Typed `CustomEvent` dispatch helper. |
| `on()` | Public | Typed DOM listener helper with listener options. |
| `host()` | Public | Current custom element plus abort signal. |
| `formControl()` | Experimental | Compiler-recognized form association helper for generated Custom Elements. |
| `Show` | Public | Static conditional rendering primitive. |
| `ComponentOptions` | Public | Flat `styles` metadata for component CSS. |

## `@iktia/runtime`

Tiny platform helper runtime for generated output: events, scheduling, and
hydration helpers. It is not a reconciler, virtual DOM, component runtime, or
framework adapter.

## `@iktia/primitives`

Experimental primitive Custom Elements authored in Iktia `.wc.tsx` and compiled
for package distribution.

| Element | Notes |
| --- | --- |
| `<iktia-button>` | Action primitive with slots, variants, parts, and `iktia-press`. |
| `<iktia-button-group>` | Oriented action grouping. |
| `<iktia-checkbox>` | ARIA checkbox primitive with experimental `FormData`, reset, and disabled fieldset support. |
| `<iktia-dropdown>` | Compiler-compatible dropdown primitive with Escape and outside pointer dismiss. |
| `<iktia-field>` | Field composition shell for labels, hints, errors, and controls. |
| `<iktia-tabs>` | Three-panel tab primitive with roving focus and orientation metadata. |
| `<iktia-toggle>` | Pressed/on-off control with experimental `FormData`, reset, and disabled fieldset support. |

## `@iktia/compiler`

```ts
type TransformResult = {
  code: string
  hasChanged: boolean
  map?: unknown
}
```

`transformComponent()` returns transformed code, optional source map data, and a
change flag. Compiler failures throw structured errors with `diagnostics[]`.

## `@iktia/vite`

`iktia(options)` installs the Vite transform and emits prerender metadata by
default for DSD workflows. Set `prerender: false` to disable metadata emission.
Vite owns module graph, chunks, CSS imports, assets, and invalidation.

## `@iktia/cli`

The `iktia` binary exposes:

| Command | Purpose |
| --- | --- |
| `iktia compile` | Transform a `.wc.tsx` component. |
| `iktia prerender` | Emit explicit Declarative Shadow DOM host HTML. |
| `iktia info` | Print compiler, package, native target, and platform details. |
